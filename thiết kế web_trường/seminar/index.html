Giới thiệu Bootstrap

Bootstrap là 1 framework HTML, CSS, và JavaScript cho phép người dùng dễ dàng thiết kế website theo 1 chuẩn nhất định, tạo các website thân thiện với các thiết bị cầm tay như mobile, ipad, tablet,...

Bootstrap bao gồm những cái cơ bản có sẵn như: typography, forms, buttons, tables, navigation, modals, image carousels và nhiều thứ khác. Trong bootstrap có thêm nhiều Component, Javascript hỗ trợ cho việc thiết kế reponsive của bạn dễ dàng, thuận tiện và nhanh chóng hơn.

Tại sao chúng ta nên sử dụng Bootstrap?

Bootstrap là một trong những framework được sử dụng nhiều nhất trên thế giới để xây dựng nên một website. Bootstrap đã xây dựng nên 1 chuẩn riêng và rất được người dùng ưa chuộng. Chính vì thế, chúng ta hay nghe tới một cụm từ rất thông dụng "Thiết kế theo chuẩn Bootstrap".

Biểu đồ thể hiện sự phân chia thị phần và tỷ lệ sử dụng các framework CSS trên các trang web


Thuận lợi khi sử dụng Bootstrap.
  • Dễ sử dụng: Bootstrap rất dễ sử dụng vì nó được xây dựng dựa trên nền tảng HTML, CSS, và JavaScript. Chỉ cần có kiến thức cơ bản về ba công nghệ này, bạn đã có thể sử dụng Bootstrap một cách hiệu quả.

  • Responsive: Bootstrap được tích hợp sẵn CSS hỗ trợ responsive, giúp hiển thị tốt trên nhiều thiết bị như iPhone, máy tính bảng, và máy tính để bàn. Tính năng này giúp tiết kiệm đáng kể thời gian trong việc tạo ra các website thân thiện với người dùng trên mọi loại thiết bị.

  • Tương thích trình duyệt: Bootstrap tương thích với hầu hết các trình duyệt phổ biến hiện nay như Chrome, Firefox, Internet Explorer, Safari, và Opera. Tuy nhiên, đối với Internet Explorer (IE), Bootstrap chỉ hỗ trợ từ IE9 trở lên, vì IE8 không hỗ trợ đầy đủ HTML5 và CSS3.

Tóm lại: Bootstrap mang lại nhiều tiện ích đáng kể trong việc phát triển giao diện web, giúp tăng hiệu suất làm việc, đảm bảo tính nhất quán và cải thiện trải nghiệm người dùng. Dưới đây là một ví dụ chứng tỏ điều đó:

Thay vì tự viết CSS để xử lý các giao diện responsive, bạn chỉ cần sử dụng các lớp sẵn có của Bootstrap như container, row, col-.

HTML và CSS thông thường:

HTML:


                    <div class="container">
                      <div class="row">
                        <div class="col-md-6 left">Left</div>
                        <div class="col-md-6 right">Right</div>
                      </div>
                    </div>

CSS:

Left
Right

                  .container {
                    width: 100%;
                    max-width: 1200px; /* Đặt chiều rộng tối đa cho vùng chứa */
                    margin: 0 auto; /* Căn giữa vùng chứa theo chiều ngang */
                    padding: 20px; /* Thêm khoảng đệm xung quanh nội dung */
                  }
                  
                  .row {
                    display: flex; /* Sử dụng flexbox để sắp xếp các cột */
                    flex-wrap: wrap; /* Cho phép các cột xuống dòng nếu cần */
                    margin: -10px; /* Sử dụng lề âm để bù đắp cho khoảng đệm cột */
                  }
                  
                  .col-md-6 {
                    width: 50%; /* Mỗi cột chiếm 50% chiều rộng của hàng */
                    padding: 10px; /* Thêm khoảng đệm giữa các cột */
                    box-sizing: border-box; /* Bao gồm phần đệm và đường viền trong chiều rộng và chiều cao của cột */
                  }
                  
                  /* Kiểu dáng cho cột Left */
                  .left {
                    background-color: #17a2b8; /* Màu nền xanh lam nhạt (info) */
                    color: white; /* Màu văn bản trắng */
                    text-align: center; /* Căn giữa văn bản */
                    padding: 15px; /* Thêm khoảng đệm bên trong cột */
                  }
                  
                  /* Kiểu dáng cho cột Right */
                  .right {
                    background-color: #dc3545; /* Màu nền đỏ (danger) */
                    color: white; /* Màu văn bản trắng */
                    text-align: center; /* Căn giữa văn bản */
                    padding: 15px; /* Thêm khoảng đệm bên trong cột */
                  }
                  
                  /* Truy vấn phương tiện cho màn hình nhỏ */
                  @media (max-width: 768px) {
                    .col-md-6 {
                        width: 100%; /* Mỗi cột chiếm toàn bộ chiều rộng trên màn hình nhỏ */
                    }
                  }

Một số chức năng

Để sử dụng Bootstrap từ một Content Delivery Network (CDN), bạn có thể thêm liên kết sau vào phần <head> của tài liệu HTML của bạn.

                
                  <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-QWTKZyjpPEjISv5WaRU9OFeRpok6YctnYmDr5pNlyT2bRjXh0JMhjY6hW+ALEwIH" crossorigin="anonymous">

2. Tạo Button đơn giản với Bootstrap Bootstrap giúp bạn tạo ra các nút (buttons) với phong cách đẹp mắt chỉ bằng cách thêm lớp CSS.

Minh họa code

                
                  <button type="button" class="btn btn-primary">Primary</button>
                  <button type="button" class="btn btn-secondary">Secondary</button>
                  <button type="button" class="btn btn-success">Success</button>
                  <button type="button" class="btn btn-danger">Danger</button>

4. Icons

Nhúng từ CDN: Chỉ cần thêm đoạn code sau vào <head> để sử dụng 

              
                  <link href="https://cdn.jsdelivr.net/npm/bootstrap-icons/font/bootstrap-icons.css" rel="stylesheet">

Dùng lớp CSS <bi> và tên biểu tượng tương ứng:

                
                <i class="bi bi-alarm"></i> <!-- Icon đồng hồ -->
                <i class="bi bi-heart-fill"></i> <!-- Icon trái tim đầy -->
                <i class="bi bi-check-circle"></i> <!-- Icon dấu tích -->

Kết quả:

Dùng các lớp kích thước mặc định của Bootstrap:

              
                <i class="bi bi-alarm fs-1"></i> <!-- Icon cỡ lớn -->
                <i class="bi bi-alarm fs-3"></i> <!-- Icon cỡ trung bình -->

Kết quả:

Chức năng đổi màu icon bằng bootstrap và css

            
              <i class="bi bi-alarm fs-1 text-primary"></i> <!-- Màu xanh -->
              <i class="bi bi-alarm fs-3" style="color: red;"></i> <!-- Màu đỏ -->
